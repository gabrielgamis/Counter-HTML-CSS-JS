# 🧮 Counter App

A simple and interactive counter application built with **HTML**, **CSS**, and **JavaScript**.  
Users can increment, decrement, and reset the counter, with support for **continuous counting by holding the buttons**.

---

## 🚀 Features

- ➕ Increment counter (hold to continuously increase)
- ➖ Decrement counter (hold to continuously decrease)
- 🔄 Reset counter to zero
- 🎨 Modern dark UI design
- ⚡ Smooth button interactions

## 📜 How It Works

- A variable `count` stores the current value.
- `setInterval` allows continuous counting when holding a button.
- `mousedown` starts the counter.
- `mouseup` stops the counter.
- `click` resets the value.
